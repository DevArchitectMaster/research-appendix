## Search Queries

### Metadata, Title, Keywords & Abstract

```bash
("Metadata": "elearning" OR
"Metadata": "e-learning" OR
"Metadata": "digital learning")

OR

("Document Title": "elearning" OR
"Document Title": "e-learning" OR
"Document Title": "digital learning")

OR

("Author Keywords": "elearning" OR
"Author Keywords": "e-learning" OR
"Author Keywords": "digital learning")

OR

("Abstract": "elearning" OR
"Abstract": "e-learning" OR
"Abstract": "digital learning")
```

> **+ manual filtering according to**  
> - [X] the **language** (*since automatic filtering via the search querie is not possible so far*)
> - [X] the corresponding **year for the publication date** (*since automatic filtering via the search querie is not possible so far*)
> - [ ] the **quality**
> - [ ] the **title**
> - [ ] the **keywords**
> - [ ] the **abstract**
> - [X] the **conclusion**
